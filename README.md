# Gestion-de-stocke
Il s'agit aussi du stock de matières premières ou de produits semi-finis. La gestion des stocks consiste à
planifier et à mettre en œuvre une méthode pour maximiser la rentabilité. Une bonne gestion de stock consiste à avoir la quantité nécessaire au bon moment
Dans ce projet j'ai creé une application descktop avec Java en utilisant l'interface graphic "swing" et aussi le SGBD mySQl et le "connector"
JDBC qui relit le mysql et java affin de stoker ,supprimer,et afficher tout les information des produit,leurs id ,et leurs quantitée.
